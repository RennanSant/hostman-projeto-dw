Ajuda

Introdução

Hostman é um software de navegador que permite gerenciar hosts na rede local. As funções disponíveis para o gerenciamento são: DESLIGAR, REINICIAR, SUSPENDER, HIBERNAR e BLOQUEAR o host cadastrado.

Hosts

Essa é a tela principal de gerenciamento, a partir desta aba é possível gerenciar os computadores com sistema Windows ou Linux. Marque as checkbox dos hosts desejados, e em seguida clique no botão do comando de gerenciamento. Estão disponíveis as opções DESLIGAR, REINICIAR, SUSPENDER, HIBERNAR e BLOQUEAR. OBS.: Algumas funções podem não estar disponíveis para algumas versões do sistema Windows ou Linux.

Armazenamento

Permite visualizar a quantidade de armazenamento utilizado, apresenta gráficos intuitivos para melhor compreensão do usuário.

Logs

Na tela de Logs é possível visualizar os dados de log em banco, como nome e data em que um usuário logou no software.

Requisitos

É necessário ter instalado e configurado um serviço de SSH no host cadastrado.

Para Linux: Instalar pacote de SSH, como o openssh-server.

Para Windows: Instalar o pacote de serviço OpenSSH, tutorial disponível em: Instalar o OpenSSH
