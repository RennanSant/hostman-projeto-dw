Projeto em JavaScript, HTML e CSS
HostMan é um software que permite gerenciar computadores em uma rede local.


Para configurar o aplicativo insira os comandos:
npm install
npm run db:create
npm start



scripts npm run:

    npm install --> instalar módulos necessários
    npm run db:create --> cria o banco de dados necessário para o funcionamento do software
    npm run dev --> rodar software em modo dev (facilita edição)
    npm start --> inicia o software
